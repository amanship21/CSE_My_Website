📚 Basic Linux Commands

pwd

Shows your current folder location.

$ pwd

ls

Lists files and folders.

$ ls

cd

Changes directory.

$ cd Downloads

mkdir

Creates a new folder.

$ mkdir linux_notes

touch

Creates a new empty file.

$ touch notes.txt

rm

Deletes files or folders.

$ rm notes.txt
